## Summary
The Windows Defender Security Center does not contain any new tools. However, it does package Windows' security tools into a neat and tidy box. With it, one can access virus and threat protection, device performance and health, firewall and network protection, app and browser control, and family options.
<br>The Windows Event Viewer is a log of what's going on under the hood, so to speak. When tracking down a system issue, seeing what the system has been up to is a good place to begin troubleshooting.
